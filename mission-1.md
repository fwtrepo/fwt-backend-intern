# Задание 1. Спроектировать БД
Разработать структуру базы данных таблиц:
- **группы** (название)
- **студенты** (ФИО, группа, дата рождения)
- **предметы** (название)
- **оценки** (студент, предмет, оценка)

Оформить с помощью инструмента [dbdesigner.net](https://www.dbdesigner.net)
